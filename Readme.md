# Forge

Forge is a full-stack real-time collaboration platform designed to help teams communicate, organize projects, and work together efficiently.

## Overview

Forge provides users with collaborative workspaces, real-time messaging, file sharing, and productivity tools in one unified platform.

## Tech Stack

### Frontend
- React
- TypeScript
- Tailwind CSS

### Backend
- Node.js
- Express
- TypeScript

### Database
- PostgreSQL
- Prisma ORM

### Real-Time Communication
- Socket.IO
